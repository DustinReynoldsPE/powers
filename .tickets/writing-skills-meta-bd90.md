# Writing-skills meta-skill


Create a guide skill for writing new Powers skills. Covers: YAML frontmatter conventions, SKILL.md structure, tk integration patterns, CSO description writing, skill types (workflow, technique, reference, domain-specific), testing methodology. Lowers the barrier for contributors.

## Design

New skill: skills/writing-skills/SKILL.md. Sections: skill anatomy (frontmatter, phases, stop gates), naming conventions, integration points (how to reference other skills, tk commands), description optimization for auto-triggering, testing approach (manual invocation checklist), anti-patterns. References existing skills as examples.

## Acceptance Criteria

Skill exists. Covers all Powers conventions. Includes examples from existing skills. Can be used to create a well-formed new skill on first attempt.
